# IBE CONSTRUCTION - Corporate Showcase Website

A high-end, minimalist corporate website for IBE CONSTRUCTION, a premier structural engineering and construction firm.

## Design Features

### Visual Identity
- **Primary Color**: Deep Navy Blue (#1B263B)
- **Accent Color**: Copper Orange (#D9480F)
- **Typography**: Inter font family with multiple weights

### Aesthetic
- Minimalist "Antigravity" style
- Fluid scroll animations
- Generous whitespace
- Structural grid layout inspired by architectural blueprints
- Glassmorphism effects on navigation

### Sections

1. **Hero Section**
   - Split-screen layout with technical grid overlay
   - Animated SVG blueprint graphic
   - Scroll indicator

2. **Services**
   - Three service cards with custom SVG icons
   - Hover animations and transitions
   - Étude de Structure, Maîtrise d'Œuvre, Conseil Technique

3. **Engineering DNA**
   - Company expertise showcase
   - Animated counters (15+ years, 200+ projects, 100% satisfaction)
   - High-quality structural imagery

4. **Portfolio**
   - Minimalist gallery grid
   - Hover overlays with project titles
   - Prepared for future project additions

5. **Contact**
   - Professional contact information
   - Sleek minimal contact form with floating labels
   - Location, phone, and email details

## How to Use

Simply open `index.html` in a modern web browser. The website works entirely with static HTML/CSS/JS.

## Browser Compatibility

- Chrome (recommended)
- Firefox
- Safari
- Edge

## Customization

To customize the website:

1. **Colors**: Edit CSS custom properties in `:root` section of `style.css`
2. **Content**: Update text directly in `index.html`
3. **Images**: Replace placeholder backgrounds with actual project images
4. **Contact Info**: Update contact details in the footer section

## Performance

- Minimal dependencies (Google Fonts only)
- Optimized animations using CSS transforms
- RequestAnimationFrame for scroll events
- Lazy-loaded animations with Intersection Observer
